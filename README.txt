Java Aufgaben für Einsteiger
============================

1. Starte die Funktion simpleThings.Main.main() im Unterverzeichnis src. Wähle im IntelliJ im Menü "Run" und dann Run 'Main'
   Im unteren Bereich des IDE (IntelliJ) erscheint eine Zahlenreihe.

2. Passe das Programm an, so dass die 7er-Reihe ausgegeben wird.

3. Passe das Programm erneut an, so dass die 7er-Reihe nur bis 70 ausgegeben wird.

4. Kommentiere in der main() Funktion den Aufruf von reihe() in Zeilen 7 und 8 aus und aktiviere den Aufruf von zahlEingeben()
   in Zeilen 11 und 12. Starte das Java Programm wie vorhin. Diesmal wirst du in der Ausgabe aufgefordert, eine Zahl einzugeben.
   Bestätige die Eingabe mit Enter.

5. Ändere das Programm ab, so dass 5 Zahlen eingelesen werden und anschliessend die Summe berechnet wird.

6. Passe das Programm an, so dass beliebig viele Zahlen eingegeben werden und die Eingabe abgeschlossen wird durch
   Eingeben einer Leerzeile (direkt Enter drücken, ohne Zahlen einzugeben).

7. Ändere das Programm erneut ab, so dass am Ende Summe und Durchschnitt ausgegeben werden.

8. Aktiviere nur die Codezeilen für Aufgabe 8. Was stimmt an der Ausgabe nicht?

9. Finde den Programmierfehler und behebe ihn.

10. Aktiviere den Code für Aufgabe 10. Probiere das Programm mit verschiedenen Eingaben aus (z.B. 7 und 12). Was geht hier schief?

11. Finde die Stelle im Code und programmiere den fehlenden Part.

12. Verändere das Programm, so dass Primzahlen bis 1000 behandelt werden können.

13. Aktiviere Aufgabe 13 in Main.main(). Du findest die Aufgabenstellung in simpleThings.Faktoren.

14. Erzeuge nun das Ergebnis für 5405400.

15. Optimiere die Primzahlenberechnung in simpleThings.Primzahlen, so dass nach Löschen der 2er-Reihe die 4er-Reihe erst gar nicht
    mehr durchlaufen wird.
